Everything Agents Need to Trust

From human intent to action execution — every step signed, scoped, and auditable.

🔐

DID Identity

W3C did:key with Ed25519. Every agent gets a cryptographic identity. No central authority needed.

📜

Verifiable Credentials

W3C VC Data Model v2.0. Scoped permissions, delegation chains, and expiry. Scope can only narrow, never widen.

🤝

Zero-Trust Handshake

5-step mutual verification with rate limiting and circuit breakers. No agent is trusted by default.

🎯

Proof of Intent

Signed, time-bounded declarations of what an agent intends to do and who authorized it. Unique to Sentinel.

🚨

Emergency Kill Switch

Revoke any agent in under 5 seconds. Cascade to all delegates. Signed proof of termination.

⭐

Reputation Scoring

Weighted scores with 90-day decay, negative vouches (2x weight), Sybil-resistant rate limiting, and quarantine.

🛡️

Content Safety

Pre-dispatch and post-response inspection. Detects prompt injection, jailbreak, PII leaks. Pluggable classifiers.

📡

Offline Mode

LRU trust caches, CRDT reputation merge, pending transaction queues. Works when connectivity is degraded.

🔌

Framework Adapters

Drop-in wrappers for LangChain, CrewAI, AutoGen, and OpenAI Agents SDK. Zero framework dependencies.

Running in 5 Lines

Add trust to your agent in under a minute.

For Developers

The SDK wires up identity, credentials, reputation, safety, offline mode, and audit logging with a single factory call.

Issue VCs, verify trust, check content safety, go offline, merge reputation — all from one object.

For MCP Servers

Drop SentinelGuard into your MCP tool handler. 10-step verification pipeline runs automatically on every tool call.

// SDK — 5 lines to trust
import { createTrustedAgent } from '@sentinel/sdk';

const agent = await createTrustedAgent({
  label: 'my-travel-bot',
  enableSafety: true,
});

// Issue credentials
const vc = await agent.issueVC({
  subjectDid: otherAgent.did,
  scope: ['flights:book'],
});

// Check content safety
const safety = await agent.checkSafety(userInput);

// Go offline with cached trust
agent.goOffline();
const decision = agent.evaluateTrust(vc);
